# VPS 1key Scripts

自用 VPS 一键脚本合集。

> 脚本主要用于个人 VPS 初始化、安全加固、服务部署和日常维护。  
> 使用前请自行 review，确认适合自己的系统环境。

## 脚本列表

| 脚本 | 说明 | 安装 |
|---|---|---|
| ssh-login-alert | SSH 成功登录 Telegram 通知 | [查看](./scripts/ssh-login-alert) |
| vps-security-baseline | VPS 基础安全加固 | 待添加 |
## 使用说明

每个脚本目录下都有独立的 `README.md` 和 `install.sh`。

建议先查看脚本内容，再执行安装。

## 注意事项

- 一键脚本默认需要 root 权限。
- 不同系统环境可能存在差异，使用前请自行测试。
